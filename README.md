<p>支持多种尺寸制作<p>
<p>支持相册上传于直接相机拍摄<p>
<p>支持多种类型的证件制作如,职业证件,公务员证件,身份证等各种类型<p>
<p>支持电子照存档等等<p>
<p>拥有小程序推荐功能,可以给其它的小程序实现引流<p>
